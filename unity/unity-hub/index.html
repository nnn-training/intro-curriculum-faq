Unity Hub のトラブル

目次

  • (1) Windows で Unity Hub にサインインできない
  • (2) Windows で Unity Editor のインストールに失敗する

(1) Windows で Unity Hub にサインインできない

Windows で Unity Hub を管理者として実行していると、サインインできないことがあるようです。
Unity Hub を一旦終了してから、通常のユーザー権限で再度実行してみてください。

(1-1) Unity Hub の終了手順

  1. タスクバーにある矢印のアイコンをクリックします。
  2. Unity Hub のアイコンを右クリックして [Unity Hub を終了] をクリックします。

(2) Windows で Unity Editor のインストールに失敗する

Windows で Unity Hub を通常のユーザー権限で実行していると、Unity Editor のインストールに失敗することがあるようです。
Unity Hub を一旦終了してから、管理者権限で再度実行してみてください。

(2-1) Unity Hub の終了手順

(1-1) Unity Hub の終了手順を参照してください。

(2-2) Unity Hub の管理者権限での実行手順

  1. タスクバーにある検索ボックスに Unity Hub と入力します。
  2. [管理者として実行] をクリックします。
